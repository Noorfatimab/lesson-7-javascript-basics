# Lesson 7 – JavaScript Basics (Button Click Interaction)

This project introduces JavaScript interactivity for the first time.

## Features
- Button with id `clickMeBtn`
- JavaScript listens for clicks
- Shows an alert when the button is clicked
- Demonstrates cause → effect using JavaScript

## Files included
- index.html – Structure of the webpage
- script.js – JavaScript logic
- README.md – Project explanation
